# Utility Scripts

## add-customer.js
Create a new customer in the database.
```bash
node add-customer.js
```

Follow the prompts to enter customer details.

## dev-server.js
Start local development server.
```bash
node dev-server.js
```

Access at `http://localhost:8787`

## test-api.js
Test API endpoints.
```bash
node test-api.js
```

Tests all endpoints and outputs results.
