## Installation

### Automatic Installation (Recommended)

1. In your WordPress admin, go to **Plugins → Add New**
2. Search for "Insertabot"
3. Click **Install Now** and then **Activate**
4. Go to **Settings → Insertabot**
5. Paste your API key (get one at [insertabot.io/signup](https://insertabot.io/signup))
6. Click **Save Changes**

Your chatbot is now live!

### Manual Installation

1. Download the Insertabot plugin ZIP from [wordpress.org/plugins/insertabot](https://wordpress.org/plugins/insertabot)
2. In your WordPress admin, go to **Plugins → Add New → Upload Plugin**
3. Upload the ZIP file and click **Install Now**
4. Click **Activate Plugin**
5. Go to **Settings → Insertabot**
6. Paste your Insertabot API key and save

## Frequently Asked Questions

### Where do I get my API key?

Create a free Insertabot account and generate an API key at:
**[https://insertabot.io/signup](https://insertabot.io/signup)**

The free plan includes everything you need to get started.

### What AI model does Insertabot use?

Insertabot uses **Meta's Llama 3.1 8B** model via Cloudflare Workers AI, providing fast, intelligent responses with enterprise-grade reliability.

### Can I train the bot on my own content?

Yes! The Pro plan includes RAG (Retrieval-Augmented Generation) support, allowing you to upload documents, FAQs, and knowledge base articles. Your chatbot will use this content to provide accurate, context-aware answers.

### Does this plugin collect personal data?

The plugin itself does not collect or store personal data. Chat interactions are processed by the Insertabot platform according to your configuration and privacy settings. All data is handled in compliance with GDPR and privacy regulations.

### Can I customize the chatbot appearance?

Absolutely! From your Insertabot dashboard, you can customize:
* Bot name and avatar
* Primary brand color
* Greeting message
* System prompt (personality)
* Widget position (bottom-right, bottom-left, etc.)
* Placeholder text

### Will this slow down my site?

No. The chatbot script loads **asynchronously** and does not block page rendering. It has zero impact on your site's performance or Core Web Vitals scores.

### Can I disable the chatbot on specific pages?

Yes. You can configure domain restrictions and page-level visibility from your Insertabot dashboard. You can also use WordPress conditional tags to control where the widget appears.

### Does it support multiple languages?

Yes! The AI model supports multilingual conversations. It will automatically respond in the language your visitors use.

### Is this plugin free?

The plugin is **100% free** to install and use. Insertabot offers:
* **Free Plan**: 1,000 messages/month, basic customization
* **Pro Plan**: Unlimited messages, RAG support, web search, priority support, advanced analytics

### How do I get support?

For plugin support, visit the [WordPress.org support forum](https://wordpress.org/support/plugin/insertabot/).
For Insertabot platform support, contact us at [insertabot.io/support](https://insertabot.io/support).
